How It Works

Four roles, one platform — seamless collaboration

01

Planner

  • Create and manage events with all details
  • Add resources and set quantities
  • Allocate resources to specific events
  • Review and respond to client requests
  • Assign staff members to events
02

Staff

  • View events assigned by planners
  • Update event status in real-time
  • Monitor allocated resources
  • Coordinate with the planner team
  • Mark events as completed or ongoing
03

Client

  • Browse all upcoming events on the platform
  • Submit custom event requests
  • Track request status updates
  • View booking details when approved
  • See allocated resources for your event
04

Vendor

  • Register as a verified resource supplier
  • List available inventory and equipment
  • Receive resource allocation requests
  • Confirm supply availability for events
  • Track delivery and pickup schedules
